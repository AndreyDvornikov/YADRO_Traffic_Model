## 1. Требования

    Linux (протестировано на Ubuntu)

    Python 3 (с pip)

    Установленная библиотека matplotlib

    C++ бинарник уже собран (лежит в корне проекта)

## 2. Подготовка
```bash

# Установить matplotlib (если ещё не)
pip install matplotlib
# Дать права на исполнение скрипту
chmod +x run_traffic_model.sh
```
## 3. Запуск
```bash

./run_traffic_model.sh <путь_к_конфигурационному_файлу> [выходной_csv]
```
Пример:
```bash

./run_traffic_model.sh test_files/equal_basic.txt result.csv
```
Если выходной файл не указан, результаты сохранятся в output.csv.

## 4. Формат входного файла (две строки)

    Первая строка: время симуляции в секундах (неотрицательное число с плавающей точкой, ≤ 3600)

    Вторая строка: описание модели и параметры
    Для равномерной: equal <период> <размер>
    Для пуассоновской: poisson <средний_интервал> <средний_размер>

## 5. Что происходит при запуске

    C++ программа генерирует трафик и сохраняет пакеты в CSV.

    Python-скрипт читает CSV, выводит характеристики (название модели, средний размер пакета, битрейт, задержку) и показывает две гистограммы.

## 6. Примечания

    В комплекте есть тестовые файлы в test_files/.
